# Historico — Respaldos y Exportaciones

Carpeta destino para todos los archivos descargados desde el dashboard del Requirement Refinator.

## Archivos esperados

| Patrón | Origen |
|--------|--------|
| `Sprint-X-respaldo-YYYY-MM-DD.md` | Respaldo HITL + EVM (exportar respaldo) |
| `Sprint-X-bitacora-pmo-YYYY-MM-DD.md` | Bitácora PMO |
| `Sprint-X-historias-refinadas-YYYY-MM-DD.md` | Resumen de HUs aprobadas |

## Cómo guardar aquí

Al hacer clic en cualquier botón de descarga del dashboard, el navegador abrirá un cuadro de diálogo nativo "Guardar como". Navega a esta carpeta para mantener el histórico ordenado.

Al importar un respaldo, el diálogo de apertura también inicia en la carpeta de documentos — navega aquí para encontrar tus respaldos previos.
